# microRNA Cancer Classification

Code and experiment materials for:

**Exploration of the Relevance of MicroRNA Signatures for Cancer Detection and Multiclass Cancer Classification**  
*IEEE Access*, 2023

## Overview

This project explores machine learning approaches for cancer detection and multiclass cancer classification using microRNA expression signatures. The work evaluates classification performance across binary and multiclass settings and includes experiments involving feature selection, PCA-based visualization, and iterative model evaluation.

## Methods

The repository includes experiments for:

- Binary cancer detection
- Multiclass cancer classification
- microRNA feature selection
- PCA visualization
- Model evaluation and result summarization
